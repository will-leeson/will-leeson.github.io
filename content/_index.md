## Research Interest

My research revolves around software verification. Software dominates the world. It's important that any software that can affect people be proven safe. Software verification can prove that some program meets a specification set by the user. This is a hard problem that becomes harder the larger and more complex a codebase gets. I aim to make this problem a more practical problem, whether that be through simplifying the problem or using the right tools for the right job.
